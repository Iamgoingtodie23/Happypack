# HAPPYPACK - Minecraft resource pack
Welcome to __Happypack!__ This simple, sleek resource pack that adds cute (and sometimes creepy!) smiles to common blocks. It is designed for people who are getting bored of normal ol' minecraft and would prefer a new, funny alternative while still maintaining a vanilla-like feel.
## Features
* Adds cute and funny faces to most blocks and entities you come across!
* Make ores really funny to find with their shiny, unique faces!
* Includes some UI elements, like hearts and menu buttons!

## Installation
1. Click on the green __code__ dropdown and select the bottom option, __Download .zip__
2. Open Minecraft and navigate to __Options > Resource Packs.__
3. Click __Open Pack Folder.__
4. Drag and drop the downloaded .zip into the folder
5. Select __HappyPack__ from the available packs list in-game and move it to the top, then click __Done.__

## Compatibility
* __Versions__: Optimized for the latest versions of Minecraft (1.20+), but works across most versions by overriding standard block and entity textures.
* __Load Order__: To ensure these bold faces override everything else, make sure HappyPack sits at the __very top__ of your Active Resource Packs list.
